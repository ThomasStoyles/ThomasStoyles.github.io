University AI Work

Please note all projects are on my Github

Module 1: Neural Networks Within my university course we had to create a neural network without using libraries. This was so that we learned how to implemenet Feed-Forward Backpropagation. The task was to create neural network that played a simple python rocket game that landed the rocket on a randomly generated lander. The network would use Feed-Forward Backpropagation with the inputs being X position to target and Y position to target. The outputs was the landers thruster and landers turning. To train the network, I played the game on the "data collection" mode. This puts the inputs and outputs into a cvs file so it can be imported into the python file using pandas. Once in the cvs file I cleaned the data and ran it through the network which I created so that we was able to get a RSME value.

Module 2: Robotics In a unstructured environment, the task involved programming the Turtlebot3 robot to complete key objectives. This included implementing a PID Controller to enable a right edge following behavior and creating a fuzzy logic controller capable of executing both right edge following and obstacle avoidance behaviors. I also had to integrate these behaviors within a unified fuzzy logic control structure, ensuring coordinated and effective functionality.

Module 4: Physics Based Games This module had us creating a game based around physics, the game I choose was an Angry birds clone. Within this game I had to code gravity, thrust amongst other, which also had their own individual physics. I added some finishing touches such as a background and score to finish.

Module 5: Data Science This module taught me how to use and display data in useful and meaningful ways. The brief for the coursework was to take weather data about Colchester and Brighton see how much renewable energy could be created through solar and wind power. With this information I had to declare how much electricity we could provide to the cities for free and for how long. There was 2 parts to this work, part one was cleaning the data and presenting it in meaningful ways through graphs and explaining how each graph helps me answer the question, along with research on how energy is produced. Part 2 was running the data through different models to see which was the most accurate and to predict the energy generated based of the research conducted in the previous part.
