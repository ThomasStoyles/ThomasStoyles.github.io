Backend Work

Please note all projects are on my Github

Project 1: Borderlands weapon generator - This project was set out by an old employer QA. The project was to create an application that generate objects ranodmly. These objects was set upon predefined rules. In this project the objects will create a random gun from the game Borderlands. This will give you a manufacturer, gun rarity and elemental damage. The project also had to be automated which was completed with Docker, Docker swarm, Jenkins and Ansible. All services was added to a VM and tested with PyTest. Please see the documentaion on the Github link for a more indepth analysis.

Project 2: Voice activated AI Here I created my own voice activated AI that follows Alexa or Siri. This is a very basic shell of what can be done with it. It takes some python libaries and used them to listen to what you are saying through your microphone. what you have said what you need to it runs this through some if statements to see if it can match a word with insturctions. E.G. If your comment has the word time in it, it will tell you the time. Of course this can be developed further to include multiple questions that use teh word "time" but this hasn't been done as of yet.

Project 3: QA CRUD Project While at Qa they asked us to create a project so that they can show potential employers our work and skills. The breif was to create a CRUD project which uses both HTML and Python to do the tasks required. These task was Create, Add, Delete and Update. Once copmleted I also added a Jenkins pipeline into the project to make sure that the program was automaticlly tested whenever there was a commit via Github. Please see the documentaion on the Github link for a more indepth analysis.

Project 4: AI Stock predictor This was one of the first AI's I created. This takes finance data from Yahoo.com as it tarining data and then predicts the stock in which you identify within the script. Here I have used Facebook which is known as FB as its stock. This is under the company variable.
